# 游戏规则
## 胜负规则
打筒子的胜负由最后的得分高低来结算，分数变动来自出牌得分、筒子/地炸的奖励分、出完牌的先手顺序会有奖罚

## 手牌
三副牌:去掉3、4、大小王，共计132 张牌。铺掉9张牌，三方每人41张牌。 铺掉的9张牌放置牌桌，好处是在最后2人对弈是，无法计算出对方的全部手牌

## 玩家
- 玩家人数3

## 出牌规则 
- 一个回合就是某个玩家出牌后，经过N轮出牌后，其他玩家没有再出牌，在有牌必出的规则下，就是其他玩家没有比该玩家当前出牌更大的牌，此时回合结束，最后出牌的玩家则为该回合的胜利者。如果该胜利者还有手牌，那么他会出牌并开启新的回合。 如果胜利者没有手牌，则由该胜利者的下一家出牌并开启回合，直到只剩下1个玩家，则游戏结束；
- 炸弹:4个或者4个以上相同数字的牌为炸弹，，讲究大打小，多打少的原则，4个2打4个A，5个打4个，6个打5个，以此类推；
- 筒子:3个相同花色同样数字的牌为筒子，筒子压炸弹，筒子也分黑桃，红桃，梅花，方块，同数字的筒子黑桃最大，方块最小。黑桃2筒子打红桃2筒子
- 地炸(三副牌):牌面最大的为地炸:相同数字的同样花色的牌各两张为地炸，地炸里面以2地炸为最大。 比如黑桃J、红桃J、梅花J、方块J各2张，一共8张牌，组成J地炸
- 单牌顺子不能出，两连对以上可以出(包括两连对)。三张牌(三副牌):可以带两张出(只能少带，不能多带)可以飞机。 飞机就是数字连续的3张，没有连续数字长度限制，可以5-2，比如3个J+3各Q，飞机中的每一组都可以带2张（可以少带）。
- 有牌必打: 出牌逻辑打的起则必须打。
- 出牌顺序: 由系统随机指定一名玩家开始
- 当牌面只剩下最后一个玩家时，游戏结束（不存在流局），计算各个玩家的得分，以及得分详情

## 计分规则
- 计分。数字5为5分，10和K均为10分，其他牌不算分。 回合胜利的玩家会得到该轮中所有5/10/K的分数。
- 三副牌:K筒子为100分，A筒子为200分，2筒子为300分，地炸奖励 400分。当局筒子压死筒子不累积分数。若A方被B方击败，B方得奖励分。 也就是当一轮牌结束时，如果结束牌是K筒子，则该玩家得分100。 需要注意的时，只有一轮牌的最后一手牌才能计算分数，加入A出牌K筒子，B出牌A筒子，假如其他玩家没有比该A筒子更大的手牌，那么B得到该200分，A不得分。
- 三副牌: 上游得分100，二游罚分40，三游罚分60

## 规则变体
- 未来还会支持多付扑克，比如4副扑克，8副扑克
- 不同人数，比如2人
- 不同铺牌张数，上面是9张
- 扑克去掉的牌不同，三副牌去掉3、4、大小王，有的还可能去掉6、7、8、9
